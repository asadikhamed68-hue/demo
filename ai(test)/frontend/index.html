About UAEU Research Assistant

UAEU Library Research Assistant is an AI-powered tool designed to help students, faculty, and researchers discover academic resources quickly and efficiently.

Features:

  • Search books from the UAEU Library catalog (OCLC WorldCat)
  • Find scholarly articles from 8 global academic databases
  • AI-curated results — picks the best articles & books for your topic
  • Personalized "Why This Article/Book" explanations for every result
  • Discover which UAEU database has the article you need
  • Direct access links to articles via DOI and database portals
  • Bilingual support (English & Arabic)

Data Sources:

  • Books: OCLC WorldCat (UAEU Library catalog)
  • Articles: CrossRef · OpenAlex · Semantic Scholar · PubMed · Europe PMC · CORE · DOAJ · OCLC WorldCat

Need human help? Click the 💬 LibChat button to chat with a librarian.
